# Security Policy

**PLEASE DON'T DISCLOSE SECURITY-RELATED ISSUES PUBLICLY.**

## Reporting a Vulnerability

### Moat (upstream tool)

Report security vulnerabilities in moat itself via the [Laravel Moat security policy](https://github.com/laravel/moat/security/policy).

### AUR package

Security issues with this AUR package (e.g. the PKGBUILD, checksums, or the release sync workflow) should be reported through GitHub's Security Advisory system at https://github.com/Jamesking56/laravel-moat-aur/security/advisories/new.
